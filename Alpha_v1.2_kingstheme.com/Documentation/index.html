We love when all updates are free. Once you’ve purchased Alpha, you’ll be able to download all future updates. If you like our theme don’t forget to rate us five stars. Good rating provides more and better updates in future!

Custom Code All theme CSS files are located in source/assets/css folder. For your CSS customization you can use source/assets/css/custom.css. Keep your custom CSS separate it makes more easier when updating the theme in future.

All Javascript files are located in source/assets/js/ folder. Also Page-level Javascript files are located in source/assets/js/pages/ folder and each page has its own Javascript file. For your Javascripts you can use source/assets/js/custom.js. Keep your custom Javascript separate it makes more easier when updating the theme in future.

Themes You can change theme very easily, by replacing color class of nav and search form. You can use any of 240+ colors from this color pallete. It's recomended using primary(without lighten/darken classes) colors for nav.
Right Sidebar Alpha framework includes layout with right sidebar, to use it you have to add right-sidebar-button class to .button-collapse button in nav on the line 79.
Hidden Sidebar By defauly Alpha uses fixed-sidebar sidebar layout, to use hidden sidebar layout you have to remove fixed-sidebar class from .mn-content and fixed class from .side-nav.
Boxed Layout You can use boxed layout by adding container class to .mn-inner.
Static Header By default Alpha uses fixed header, but if you want to make it static, you have to remove navbar-fixed class from .mn-header.
New Page It's very easy to create new page with Alpha.

First of all you have to copy layout-blank.html file and rename it as you want, then you have to choose layout and theme color and change them as explained previously. then you can start creating content of page, you can use all components that are shown in the demo of Alpha. Note, that all content should be placed in .mn-inner to fit the page correctly.

If you will use component that needs some JS codes which are included in component demo page at the end of the HTML body tag, you can just include the same JS file. If you will use component that needs an extra plugin to work, you have to include files of this plugins as they are included in the demo page.
